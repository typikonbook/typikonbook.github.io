Typikon

Typikon 让您可以使用 Markdown 编写您的在线书籍。

介 绍

Typikon 的名字源自 Typikon 书籍，它是一个类似于 mdbook 和 gitbook 的静态网站渲染工具，但它专注于将 Markdown 渲染成在线书籍，比其他工具更易于使用。

GitHub：github.com/auula/typikon 🌟

特 点

  • 支持暗黑和明亮主题
  • 支持自定义 CSS 和 JS
  • 仅关注目录和文件映射
  • 渲染静态文件以便于部署
